Introduction to Alpha

Alpha is a statically typed, compiled language designed for building scalable and maintainable software. It combines the performance of C with the expressiveness of modern scripting languages.

Note: Alpha is currently in version 1.0.0. Features may change in future releases.

Installation

To install Alpha, download the binary for your operating system from the Download page or use the terminal:

curl -s https://get.alpha-lang.org | bash

Your First Program

Create a file named main.al and paste the following code:

main.al
import alpha.io

func main() {
  io.println("Hello, World!")
}

Variables & Types

Alpha is strongly typed. You can declare variables using var (mutable) or let (immutable).

types.al
var count: int = 10
let name: string = "Alpha"
var isActive: bool = true

// Type inference is also supported
var score = 95.5 // inferred as float

Control Flow

Standard if, else, and loops work as expected.

flow.al
if (score > 90) {
  io.println("Excellent!")
} else {
  io.println("Keep trying.")
}

for (i in 0..10) {
  io.print(i)
}

Functions

Functions are first-class citizens in Alpha.

func.al
func add(a: int, b: int) -> int {
  return a + b
}

Standard IO

The alpha.io module provides basic input and output operations.

  • io.print(msg): Prints to stdout without newline.
  • io.println(msg): Prints to stdout with newline.
  • io.readln(): Reads a line from stdin.
