Hello World

The standard "Hello World" program in Alpha.

main.al
package main

import alpha.io

func main() {
  io.println("Hello, Alpha!")
}

Control Flow

Alpha uses standard if, else, and for structures.

flow.al
let num = 15
if (num > 10) {
  io.println("Greater than 10")
} else {
  io.println("Less than or equal to 10")
}

for (i in 0..10) {
  io.print(i)
}

Functions

Functions are first-class citizens in Alpha.

func.al
func add(a: int, b: int) -> int {
  return a + b
}

Standard IO

The alpha.io module provides basic input and output operations.

  • io.print(msg): Prints to stdout without newline.
  • io.println(msg): Prints to stdout with newline.
  • io.readln(): Reads a line from stdin.
