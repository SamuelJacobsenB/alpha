Simple HTTP Server

Create a web server in just a few lines of code using the standard library.

server.al
import alpha.net.http

func main() {
  var server = http.newServer(8080)

  server.handle("/", (req, res) => {
    res.send("Welcome to Alpha!")
  })

  server.start()
}

Recursive Fibonacci

Demonstrating recursion and type definitions.

math.al
func fib(n: int) -> int {
  if (n <= 1) return n
  return fib(n - 1) + fib(n - 2)
}

print(fib(10)) // Output: 55
